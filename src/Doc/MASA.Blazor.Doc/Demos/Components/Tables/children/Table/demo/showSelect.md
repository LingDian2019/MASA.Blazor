## zh-CN

show-select 属性将在默认表头中渲染一个复选框以切换所有行是否被选择，同时也为每个默认行渲染一个复选框。 你可以分别使用 header.data-table-select 和 item.data-table-select 的插槽来自定义。 你还可以使用 single-select 属性，指定能同时选择多行还是只能选择一行。
